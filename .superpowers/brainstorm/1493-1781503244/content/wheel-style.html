中午转盘 — 视觉风格方向

旧版转盘问题：颜色太暗、文字跟着转看不清、扁平无层次。以下是三种改进方向。

A

克莱因蓝渐变扇区 · 轻量优雅

扇区颜色用克莱因蓝 + 浅蓝 + 白色交替，浅底深字，文字不随转盘旋转（固定在扇区上方用 pointer-events 穿透）。外圈加 tick marks，中心放 logo/emoji。

B

高对比暗色扇区 · 视觉冲击

每个扇区用不同的高饱和颜色，白字，扇区间加白色分隔线。模仿真实幸运转盘的华丽感。文字仍固定在扇区中段。

C

极简刻度环 · 指针旋转

转盘本身不旋转，而是一个发光指针在外圈旋转。选项文字排列在环内，选中项高亮放大。转盘静态 + 指针动态。

关键决策点

扇区配色 方案 A：克莱因蓝系渐变（白/浅蓝交替）| 方案 B：多彩高饱和 | 方案 C：无扇区，纯刻度环
文字策略 文字固定在扇区外侧不随转盘旋转（用 absolute + 反旋转抵消），还是放弃标签只在外圈显示名字？
指针样式 CSS 三角箭头 | SVG 拟物指针 | 发光光柱
开奖仪式感 转盘停止后 → 中心弹出结果卡片 + 缩放动画 | 粒子/光晕扩散 | 底部抽屉滑入
